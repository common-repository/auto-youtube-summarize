== Description ==

The Automatic Youtube Video Posts Plugin with Youtube Data API and wp_cron. You can use search functions such as specifying a name, search order.And can use post comment customization.

If you use Youtube Data API for the first time, rest assured that there is a manual.

This plugin:

* You can set keywords by yourself.
* You can set your preferences, such as acquisition date and priority order.
* Manual complete. We also offer unlimited support after purchase.
* Customize the contents of the post on the summary site to your own taste. Go to the original site!
* Affiliates, adsenses and other advertising revenue sources!
* Automatically import YouTube videos into WordPress Posts from a specified Keyword.
* Uses YouTube Data API v3.
* Add a youtube video meta information.Use replacement character.
* Post automatically with wp_cron.
* Add-ons : Add search order customize.
* Add-ons : Option comments in posts.

== Frequently Asked Questions ==

= How do I set up a Google API Key? =

Please read installation guide.After you install plugin,you can view the guide in setting page.

= Does this plugin do duplicate post ? =

No.Same title videos post once.

= Can I customize post content? =

Yes. You can customize post to yours. This plugin is ready for replacement character as "High rating", "Low rating", "Desctiption", "Channel Title", and so on.

= When wp_cron run? =

Twice at a daily.

== Installation ==

1. Upload the `auto-youtube-summarize` directory to your `/wp-content/plugins/` directory
2. Activate the plugin through the 'Plugins' menu in WordPress
